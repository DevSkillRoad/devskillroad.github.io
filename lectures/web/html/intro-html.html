Введение в HTML

📝 Первое знакомство с языком гипертекстовой разметки

HTML — это основа любого сайта. Именно с него начинается путь веб-разработчика.

Что такое HTML?

HTML (HyperText Markup Language) — язык гипертекстовой разметки.

Разберём это название по частям:

  • HyperText (гипертекст) — это текст с интерактивными ссылками, которые позволяют переходить между фрагментами документа (страницы) или разными документами.
  • Markup (разметка) — система специальных меток, описывающих структуру и форматирование содержимого.
  • Language (язык) — набор правил и синтаксиса для записи этих меток.

Как работает HTML?

Вы пишете текст и специальные метки (теги), а браузер читает их и показывает страницу так, как вы задумали. Теги объясняют браузеру: «это заголовок», «это абзац», «а это ссылка».

Инструменты для работы с HTML

Вам понадобятся три простых инструмента:

  • Текстовый редактор — редактор для работы с чистым текстом. Подойдёт любой, но лучше сразу взять удобный:
    • Visual Studio Code (VSCode) — самый популярный, бесплатный, с кучей дополнений
    • Sublime Text — быстрый и красивый
    • Notepad++ — простой и лёгкий
  • Браузер — для просмотра и тестирования результатов работы (Chrome, Firefox, Safari, Edge).
  • Валидатор — сервис для проверки синтаксиса и выявления ошибок. Позже я покажу, где их искать.
