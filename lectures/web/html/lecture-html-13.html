Спецсимволы в HTML

&nbsp; &lt; &gt; &copy; &mdash; и другие

В HTML есть символы, которые нельзя написать просто так. Например, угловые скобки используются для тегов, а амперсанд — для начала самой спецпоследовательности. Для их отображения применяют HTML-сущности (спецсимволы).

1. Основные спецсимволы

Результат Как написать в коде Описание
< &lt; Меньше (less than) — нужно для показа тегов на странице
> &gt; Больше (greater than) — нужно для показа тегов на странице
& &amp; Амперсанд (ampersand) — нужен, чтобы написать сам спецсимвол
  &nbsp; Неразрывный пробел — запрещает перенос строки в этом месте
© &copy; Знак копирайта (copyright)
® &reg; Зарегистрированная торговая марка (registered)
™ &trade; Торговая марка (trademark) — ™
— &mdash; Длинное тире (em dash) — используется в тексте как пауза
– &ndash; Короткое тире (en dash) — для диапазонов (10–20 км)
« &laquo; Левая двойная кавычка-ёлочка («)
» &raquo; Правая двойная кавычка-ёлочка (»)
… &hellip; Многоточие (…)

2. Когда это нужно

Пример 1: показать код на странице

Если вы хотите показать пример тега в своей лекции, нужно заменять угловые скобки на &lt; и &gt;.

&lt;p&gt;Это абзац&lt;/p&gt;

Что увидит пользователь:

<p>Это абзац</p>

Пример 2: неразрывный пробел

Обычный пробел браузер может перенести на новую строку. Неразрывный пробел &nbsp; запрещает перенос в этом месте.

Это &nbsp;&nbsp;&nbsp; три неразрывных пробела.

Что увидит пользователь:

Это     три неразрывных пробела.
📌 Неразрывный пробел часто используют в ФИО (Иван&nbsp;Петров) или между числом и единицей измерения (10&nbsp;кг).

Пример 3: типографские символы

Длинное тире, кавычки-ёлочки, знаки копирайта делают текст профессиональнее.

&copy; 2025 DevSkillRoad&nbsp;&mdash;&nbsp;Все права защищены.

Что увидит пользователь:

© 2025 DevSkillRoad — Все права защищены.

3. Полные справочники

В интернете есть полные таблицы всех HTML-сущностей (например, на справочных сайтах). Если вам понадобится редкий символ — вы легко его найдёте по запросу «HTML спецсимволы таблица».

4. Что нельзя делать

  • Использовать обычные угловые скобки для показа кода — браузер примет их за теги.
  • Злоупотреблять &nbsp; вместо отступов — для отступов есть CSS.
  • Заменять все кавычки на спецсимволы — обычные двойные и одинарные работают нормально.
📓 Из тетради: «Спецсимволы нужны нечасто, но когда нужны — без них никуда. Запомните &lt;, &gt;, &amp;, &nbsp; и &copy; — остальные можно посмотреть в справочнике».
